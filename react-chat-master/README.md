# React Chat
## A simple chat with React & Socket.IO

### Start the server API by running:
```sh
cd chat-server
yarn install
yarn run compile
yarn run start
```

### Start the client by running:
```sh
cd chat-client
yarn install
yarn run start
```

### Open the Chat App
- Go to `http://localhost:3000`.
